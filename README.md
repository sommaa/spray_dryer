# spray_dryer
Mathematical modeling of a spray dryer in matlab and OpenFoam (only flow-dynamic).

Simulation details:
- 2Mln mesh cells;
- k-omega-SST RAS model;
- inlet Q=20Kg/s;
- outlet Pout = 0;
